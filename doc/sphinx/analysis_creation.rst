*****************
Analysis creation
*****************


Introduction
============

Analysis has predefined :func:`~i3skyllh.analyses.IC170922A_wGFU.analysis.create_analysis` function. To customize the analysis we need to:

#. Set `sample_seasons` variable.
#. Set `source` variable.
#. Create `analysis` instance by running :func:`~i3skyllh.analyses.IC170922A_wGFU.analysis.create_analysis` function which acts as constructor.
